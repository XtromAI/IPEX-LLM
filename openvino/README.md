## Overview

OpenVINO (Open Visual Inference and Neural network Optimization) toolkit is Intel's solution for optimizing and deploying AI inference across various Intel hardware platforms.

### Target Hardware
- Intel CPUs (various generations)
- Intel integrated GPUs
- Intel discrete GPUs (Arc series)
- Intel Neural Compute Stick
- Intel Movidius VPUs

### Supported Model Formats
- OpenVINO IR (Intermediate Representation)
- ONNX
- TensorFlow
- PyTorch (via conversion)
- PaddlePaddle
